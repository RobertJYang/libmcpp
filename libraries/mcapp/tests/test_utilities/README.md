## 概述

本文档描述了测试工具模块。该模块提供了测试辅助功能，包括测试基类、资源管理、D-Bus 守护进程管理等。

## 文件列表

测试工具模块包含以下文件：

1. `dbus_daemon_manager.cpp` - D-Bus 守护进程管理器实现
2. `resource_manager.cpp` - 资源管理器实现

## 用途

这些文件为测试用例提供通用的测试基础设施和工具函数，不包含测试用例本身。
